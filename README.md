# Mood

Women's vaginal health supplement brand. Core product is an oral probiotic capsule (5B CFU probiotic blend + XOS prebiotic + cranberry powder) marketed for vaginal flora support, pH balance, and urinary wellness.

All marketing and copy uses structure/function claim language only — no disease treatment claims.
